# PRD - KORTEX (Product Requirements Document)

## 1. Product Overview
Kortex is a global platform connecting Turkish diaspora worldwide.

## 2. Problem
- Lack of centralized diaspora information
- Difficulty accessing local Turkish services
- Fragmented communities

## 3. Solution
A unified platform combining:
- Listings (businesses, professionals)
- Communities
- Events
- AI-powered search

## 4. Core Features
- User roles (individual, business, community, moderator)
- Listing system
- Claim system
- AI search
- Map integration
- Review & rating system

## 5. Success Metrics
- User growth
- Retention rate
- Revenue generation

## 6. MVP Scope
- Germany-first launch
- Core listings + search
- Basic moderation
- Authentication (Google/Apple)

## 7. Future Scope
- Mobile app
- Advanced AI recommendations
- Monetization features
