## BÖLÜM E: İnsan Kaynakları (HR) Modülü — CV Upload

### Hedef
Kullanıcıların CV yükleyebileceği ve listelenebileceği bir İnsan Kaynakları sayfası oluşturmak. Supabase Storage + DB referans mimarisi.

### Mimari
- Dosyalar → **Supabase Storage (private bucket)**
- Veritabanında → sadece file_path + metadata
- CV erişimi → **signed URL** (5-10 dakika geçerli)

### Storage Yapısı

**Bucket: `cv-files`** (private)
```
cv-files/{user_id}/{uuid}.pdf
```
- Supabase Dashboard > Storage > New Bucket > `cv-files` (Private)
- Proje URL: `https://supabase.com/dashboard/project/injprdrsklkxgnaiixzh/storage/files/buckets`

### Veri Modeli

**Tablo: `user_cvs`**

| Alan | Tür | Açıklama |
|------|-----|----------|
| `id` | UUID (gen_random_uuid()) | PK |
| `first_name` | TEXT NOT NULL | İsim |
| `last_name` | TEXT NOT NULL | Soyisim |
| `role` | TEXT | Görev / Pozisyon |
| `file_path` | TEXT NOT NULL | Storage'daki dosya yolu |
| `file_name` | TEXT NOT NULL | Orijinal dosya adı |
| `created_at` | TIMESTAMPTZ (now()) | Oluşturma tarihi |

### Adımlar

#### E.1 Supabase Migration
- **Dosya:** `supabase/migration_008_user_cvs.sql` (yeni)
- `user_cvs` tablosu oluştur
- RLS: CV'ler herkese listelenebilir ama signed URL olmadan dosyaya erişilemez

#### E.2 Supabase Storage Bucket
- `cv-files` bucket'ı Supabase Dashboard'dan manuel oluşturulacak (Private)
- RLS policy: authenticated users okuyabilir (veya "Allow all" — dashboard internal kullanım)

#### E.3 CV Type Definitions
- **Dosya:** `src/lib/cv-items.ts` (yeni)
- `CvItemRow`, `CvItem`, `CvFormState` interfaceleri
- Helper fonksiyonları

#### E.4 CVManager Component
- **Dosya:** `src/components/hr/CvManager.tsx` (yeni)
- `getSupabaseBrowserClient()` ile CRUD + Storage upload
- **Upload Akışı:**
  1. Form: İsim, Soyisim, Görev, CV dosya input
  2. Dosya `cv-files` bucket'ına `supabase.storage.from('cv-files').upload(path, file)` ile yüklenir
  3. Dönen path DB'ye `user_cvs` tablosuna kaydedilir
  4. Başarı mesajı gösterilir
- **Listeleme:**
  - Tablo: İsim Soyisim, Görev, 2 buton (View CV, Download CV)
  - Her buton tıklamasında `supabase.storage.from('cv-files').createSignedUrl(path, 300)` ile signed URL üretilir
  - 5 dakika (300 saniye) geçerli link
- Desktop: tablo, Mobile: kartlar
