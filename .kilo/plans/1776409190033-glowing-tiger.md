## 1. `meeting_notes` Supabase Tablosu Oluşturma

**Durum:** Yapılmadı - Veriler şu an `src/lib/meeting-notes-data.ts` dosyasında hardcoded.

**Yapılacaklar:**
- `supabase/migrations/` altına yeni migration dosyası oluştur
- `meeting_notes` tablosu oluştur:
  ```sql
  CREATE TABLE meeting_notes (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    title TEXT NOT NULL,
    content TEXT NOT NULL,
    date DATE NOT NULL,
    category TEXT NOT NULL,        -- 'rezervasyon' | 'kullanici' | 'audit' | 'veritabani' | 'mvp' | 'reklam'
    color TEXT NOT NULL,           -- '#EA4335' | '#34A853' | '#4285F4' | '#FBBC04' | '#A142F4' | '#00897B'
    source TEXT NOT NULL,          -- 'toplanti1' | 'toplanti2' | 'toplanti3' | 'whatsapp'
    subcategory TEXT,
    tags TEXT[] DEFAULT '{}',
    sort_order INTEGER DEFAULT 0,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
    updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
  );
  ```
- Mevcut hardcoded verileri seed migration olarak ekle (~70+ madde)
- `src/lib/meeting-notes-data.ts` dosyasını Supabase fetch'e dönüştür

## 3. Koyu Tema (Dark Mode) Desteği

**Durum:** Yapılmadı - Sadece açık tema mevcut.

**Yapılacaklar:**
- Tailwind `darkMode` konfigürasyonunu `class` stratejisi ile aktif et
- `tailwind.config.ts` dosyasında `darkMode: 'class'` ekle
- Global CSS'de `:root` ve `.dark` CSS değişkenlerini tanımla:
  - Arka plan renkleri (dark: `#0f172a`, `#1e293b`)
  - Metin renkleri (dark: `#e2e8f0`, `#94a3b8`)
  - Kart renkleri (dark: `#1e293b` border `#334155`)
  - Accent renkleri (kategoriler için koyu varyantlar)
- Tüm bileşenlerde `dark:` prefix'li Tailwind sınıfları ekle
- Theme toggle bileşeni oluştur (`ThemeToggle.tsx`) - güneş/ay ikonu
- `localStorage`'a tema tercihini kaydet
- `<html>` elementine `class="dark"` ekleme/çıkarma mantığı
